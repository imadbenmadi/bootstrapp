📐 Grid System

Bootstrap uses a 12-column grid layout for responsive design.

25% - col-md-3
25% - col-md-3
25% - col-md-3
Code:
<div class="row">
  <div class="col-md-4">25%</div>
  <div class="col-md-4">25%</div>
  <div class="col-md-4">25%</div>
</div>

Responsive Breakpoints:

  • col-xs - Extra small (< 576px)
  • col-sm - Small (≥ 576px)
  • col-md - Medium (≥ 768px)
  • col-lg - Large (≥ 992px)
  • col-xl - Extra large (≥ 1200px)
  • col-xxl - Extra extra large (≥ 1400px)
