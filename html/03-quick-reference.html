1. Getting Started - Setup

Option 1: CDN (Quickest)

<!DOCTYPE html>
<html>
<head>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1 class="text-center p-5">Hello Bootstrap!</h1>
    
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Option 2: NPM (Professional)

npm install bootstrap
// In your JavaScript file:
import 'bootstrap/dist/css/bootstrap.css'
import * as bootstrap from 'bootstrap'
💡 Tip: For learning and quick projects, use CDN. For large projects, use NPM for customization.

3. Spacing & Utilities

Margin & Padding (m/p + direction + size)

/* Direction: t(top), b(bottom), s(start/left), e(end/right), x(horizontal), y(vertical) */
.mt-3        /* margin-top: 1rem */
.mb-5        /* margin-bottom: 3rem */
.px-2        /* padding-left & right: 0.5rem */
.py-4        /* padding-top & bottom: 1.5rem */
.m-auto      /* margin: auto (center) */
.mx-auto     /* horizontal margin: auto */

Common Utilities

.bg-primary, .bg-success, .bg-danger Background colors
.text-primary, .text-muted, .text-center Text colors & alignment
.d-flex, .justify-content-center Flexbox layout
.display-1 to .display-6 Large headings
.fs-1 to .fs-6 Font sizes
.fw-bold, .fw-normal, .fst-italic Font weight & style
✓ Best Practice: Use utility classes for quick styling instead of writing custom CSS for common properties like margins, padding, and colors.

Best Practices

1. Mobile First: Use breakpoints from smallest to largest. Start with .col-12 (mobile), then add .col-md-6 (tablet).
2. Semantic HTML: Use proper HTML tags (<header>, <nav>, <main>, <footer>) with Bootstrap.
3. Use Utility Classes: Bootstrap's utility classes are powerful. Use them before writing custom CSS.
4. Consistency: Keep color schemes, spacing, and typography consistent throughout the site.
5. Accessibility: Always include alt text for images, use proper heading hierarchy, and ensure sufficient color contrast.
