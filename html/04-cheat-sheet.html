📋 Bootstrap 5 Cheat Sheet

Quick reference for common Bootstrap classes and components

Grid System
Containers
.container - Max width 1200px
.container-fluid - Full width
.container-xl - Max width 1320px
Columns
.col - Auto width
.col-md-6 - 50% on medium
.col-lg-4 - 33% on large
Typography
Headings
.h1 to .h6 - Heading styles
.display-1 - Extra large
.display-6 - Small display
Text Utilities
.fw-bold - Font weight bold
.fst-italic - Italic text
.text-muted - Gray text
Colors & Backgrounds
Text Colors
.text-primary primary
.text-success success
.text-danger danger
.text-warning warning
.text-info info
Background Colors
.bg-primary primary
.bg-success success
.bg-danger danger
.bg-warning warning
.bg-info info
Spacing (Margin & Padding)
Format: {property}{sides}-{size}
Property: m (margin), p (padding)
Sides: t, b, s, e, x, y (or none for all)
Sizes: 1-5 (0.25rem to 3rem), auto

.mt-3 - margin-top: 1rem
.mb-5 - margin-bottom: 3rem
.px-2 - padding-left & right: 0.5rem
.m-auto - margin: auto
Flexbox & Display
Display & Positioning
.d-flex - Display flex
.d-grid - Display grid
.d-none - Display none
.d-md-block - Block on medium+
Flex Utilities
.justify-content-center
.align-items-center
.flex-wrap
.gap-3 - Spacing between items
Buttons
Button Styles
.btn .btn-primary
.btn .btn-outline-primary
.btn .btn-lg - Large button
.btn .btn-sm - Small button
Button States
.disabled - Disabled state
.active - Active state
.btn-group - Button group
.dropdown-toggle - Dropdown button
Cards
.card - Card container
.card-header - Header section
.card-body - Main content
.card-footer - Footer section
.card-title, .card-text, .card-img-top
Forms
Form Elements
.form-label - Label styling
.form-control - Input/textarea
.form-select - Select dropdown

Form Layout
.form-floating - Floating labels
.input-group - Grouped inputs
.was-validated - Validation
.is-invalid - Error state
Navigation Bar
.navbar - Navigation container
.navbar-brand - Logo/brand
.nav - Navigation list
.navbar-expand-lg - Responsive collapse
.navbar-dark, .navbar-light - Theme
.sticky-top - Sticky navigation

.modal-dialog - Dialog box

Alerts

Tables
