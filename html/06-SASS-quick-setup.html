⚡ SASS Quick Setup (3 Minutes)

Get started with SASS/SCSS in 3 different ways - pick what works for you!

🚀 Option 1: Easiest (CDN + Custom CSS)

⏱️ 2 Minutes | ✅ No Installation

If you don't have Node.js installed, this is your fastest option!

1

Create Your CSS File

Create custom.css in your project folder: 

/* custom.css */
:root {
  --primary-color: #0066cc;
  --secondary-color: #FF6B6B;
  --success-color: #28a745;
}

body {
  --bs-body-font-size: 16px;
  color: var(--primary-color);
}

.btn-custom {
  background-color: var(--primary-color);
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
}
2

Link in Your HTML

<!DOCTYPE html>
<html>
<head>
    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
    
    <!-- Your Custom CSS -->
    <link rel="stylesheet" href="custom.css">
</head>
<body>
    <button class="btn-custom">My Custom Button</button>
    
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
✅ Done! You now have Bootstrap customization without SASS compilation. Use CSS variables for colors and styles.
✓ Pros
  • No installation needed
  • Works immediately
  • Easy to understand
✗ Cons
  • No variables/mixins
  • More repetitive code
  • No nesting support

🎯 Option 2: SASS Compiler (Best Balance)

⏱️ 5 Minutes | ✅ Recommended

Using Dart Sass - the official SASS compiler. Works on Windows, Mac, Linux.

1

Install Node.js (if not installed)

Download from nodejs.org (LTS version)

2

Install Sass

Open Terminal/PowerShell in your project folder and run: 

npm install -g sass

Verify installation:

sass --version
3

Create Your SCSS File

Create main.scss:

// Define variables
$primary-color: #0066cc;
$secondary-color: #FF6B6B;
$base-font-size: 16px;

// Import Bootstrap
@import "../node_modules/bootstrap/scss/bootstrap";

// Add your styles
body {
  font-size: $base-font-size;
  color: $primary-color;
}

.btn-custom {
  background-color: $secondary-color;
  padding: 10px 20px;
  border-radius: 4px;
  
  &:hover {
    background-color: darken($secondary-color, 10%);
  }
}
4

Compile SCSS to CSS

Run in Terminal:

sass main.scss main.css

Or use Watch Mode (auto-compile when you save): 

sass --watch . --format=compressed
5

Link in HTML

<link rel="stylesheet" href="main.css">
✅ Done! You now have a professional SASS setup with full Bootstrap customization!
✓ Pros
  • Full SASS features
  • Bootstrap customization
  • Auto-compile watch mode
  • Professional setup
✗ Cons
  • Requires Node.js
  • Command line usage
  • Terminal skills needed

💻 Option 2B: VS Code Extension (Easiest Compilation)

⏱️ 5 Minutes | ✅ GUI-based

Use VS Code's Live Sass Compiler extension - no terminal commands!

1

Install Extension

In VS Code:

  • Go to Extensions (Ctrl+Shift+X)
  • Search "Live Sass Compiler"
  • Install by Glenn Marks
2

Create SCSS File

Name your file main.scss (extension matters!) 

$primary: #0066cc;

body {
  background-color: $primary;
}
3

Start Watching

Click "Watch Sass" in the bottom status bar

It will automatically compile .scss files to .css whenever you save!

✅ Done! Extension will auto-compile and even open preview in browser.
✓ Pros
  • No terminal needed
  • Automatic compilation
  • Built-in live preview
  • Very user-friendly
✗ Cons
  • Only works in VS Code
  • Extension dependency
  • Bootstrap import may need npm

Common SASS Commands

Basic Commands
// Single compile
sass input.scss output.css

// Watch folder
sass --watch .

// Compressed output
sass --watch . --style=compressed

// Check version
sass --version
NPM Scripts Setup

Add to package.json "scripts":

"sass": "sass",
"sass:watch": "sass --watch .",
"sass:build": "sass . --style=compressed"

Then run:

npm run sass:watch

Troubleshooting

❌ "sass command not found"
Run: npm install -g sass and restart your terminal.
❌ Bootstrap import not working
Make sure Bootstrap is installed: npm install bootstrap
❌ CSS file not updating
Make sure watch mode is running: check for "Watch Sass" button or run sass --watch .
❌ VS Code Extension not compiling
Try: Disable and re-enable the extension, or save the .scss file again.
