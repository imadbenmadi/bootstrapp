Complete reference guide for all Bootstrap 5.3 helpers and utilities

HELPERS

1. Clearfix

Clear floated content within a parent element.

Usage: .clearfix
Purpose: Prevents floated elements from breaking layout 
<div class="clearfix">
    <img src="image.jpg" class="float-start">
    Text wraps around image, then clears
</div>

7. Text Truncation

Truncate and clamp text with ellipsis.

Single line (text-truncate):
This is a very long text that will be truncated with an ellipsis because it exceeds the container width.
Multi-line (line-clamp):

This is a longer text that will be clamped to exactly 2 lines. Any text beyond the second line will be hidden with an ellipsis. This is useful for descriptions and summaries. 

<!-- Single line truncation -->
<div class="text-truncate">Long text here...</div>

<!-- Multi-line clamp (2 lines) -->
<p style="--bs-line-clamp: 2;">
    Long paragraph text that will be clamped to 2 lines...
</p>

3. Display

Control how elements are displayed.

Display Values:
  • d-none - display: none
  • d-inline - inline
  • d-inline-block - inline-block
  • d-block - block
  • d-flex - flexbox
  • d-grid - grid
Responsive:
  • d-md-none - hidden on md+
  • d-lg-block - block on lg+
  • d-print-none - hidden when printing
  • d-print-block - show on print 
<div class="d-none">Hidden</div>
<div class="d-none d-md-block">Show on medium+</div>
<div class="d-md-none">Hide on medium+</div>
<div class="d-print-none">Hide when printing</div>

9. Spacing (Margin & Padding)

Control margins and padding with consistent spacing scale.

Format: {property}{side}-{size}
  • m = margin, p = padding
  • t = top, b = bottom, s/start = left, e/end = right
  • x = horizontal, y = vertical
  • 0-5 = size scale (0=0, 1=0.25rem, 5=3rem)
Examples:
  • m-3 = margin 1rem all sides
  • mt-5 = margin-top 3rem
  • px-2 = padding left/right 0.5rem
  • ms-auto = margin-left auto (right align) 
<div class="m-3 p-4">Margin 1rem, Padding 1.5rem</div>
<div class="mt-5 mb-2">Top margin 3rem, bottom 0.5rem</div>
<div class="mx-auto">Centered with left/right auto margin</div>
<div class="ms-auto">Right aligned</div>

11. Visibility

Control element visibility while maintaining document flow.

Visibility Classes:
  • visible - visibility: visible
  • invisible - visibility: hidden (keeps space)
Display vs Visibility:
  • d-none - removes from flow
  • invisible - hidden but keeps space
  • visually-hidden - screen reader only 
<div class="invisible">Hidden but takes space</div>
<div class="d-none">Hidden and no space</div>
<div class="visually-hidden">Screen readers only</div>

Best Practices

DO
  • Use utilities for quick styling
  • Combine related utilities logically
  • Use responsive variants for layouts
  • Stack utilities for complex effects
DON'T
  • Avoid utility bloat on single elements
  • Don't mix container and component utilities
  • Avoid replacing semantic HTML
  • Don't ignore responsive breakpoints
